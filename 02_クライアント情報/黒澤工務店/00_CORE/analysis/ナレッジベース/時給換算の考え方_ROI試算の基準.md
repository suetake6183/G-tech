# 時給換算の考え方とROI試算の基準

**作成日**: 2025年11月
**作成者**: 末武修平
**目的**: 投資対効果試算の透明性と妥当性の確保

---

## 前提：なぜ時給換算が重要か？

業務効率化や組織変革の投資対効果を測る際、
**「工数削減」を金額換算する**必要があります。

その際の時給設定が不適切だと：
- ROIが過小評価される → 投資判断を誤る
- または過大評価される → 期待外れになる

**透明性のある、妥当な基準が必要**です。

---

## 時給換算の一般的な算出方法

### 基本式
```
時給 = 年収 ÷ 年間労働時間
```

### 年間労働時間の標準
- **フルタイム正社員**: 約2,000時間/年
  - 計算根拠：250日（週5日×50週）× 8時間 = 2,000時間
  - （年間休日115日を想定）

### 例
| 年収 | 時給換算 |
|------|---------|
| 300万円 | 1,500円 |
| 400万円 | 2,000円 |
| 500万円 | 2,500円 |
| 600万円 | 3,000円 |
| 800万円 | 4,000円 |
| 1,000万円 | 5,000円 |
| 1,500万円 | 7,500円 |
| 2,000万円 | 10,000円 |

## 建設業界の年収相場（参考）

### 工務店経営者（従業員20-30名規模）
**年商規模**: 5-15億円（推定）

**社長の年収相場**:
- 控えめ: 800-1,200万円
- 一般的: 1,200-2,000万円
- 成功企業: 2,000-3,000万円以上

**時給換算**:
- 控えめ: 4,000-6,000円
- 一般的: 6,000-10,000円
- 成功企業: 10,000-15,000円

### 実務責任者（勤務10年、幅広い業務担当）
**年収相場**:
- 控えめ: 500-700万円
- 一般的: 700-900万円
- 高待遇: 900-1,200万円

**時給換算**:
- 控えめ: 2,500-3,500円
- 一般的: 3,500-4,500円
- 高待遇: 4,500-6,000円

## 黒澤工務店における時給設定（修正版）

### 修正前（誤り）
| 役職 | 時給 | 問題点 |
|------|------|--------|
| 黒澤社長 | 2,500円 | 年収500万円相当（過小評価） |
| 亜希さん | 2,500円 | 年収500万円相当（過小評価） |

**問題**:
- 従業員20-30名規模の工務店社長が年収500万円はあり得ない
- 実務責任者（10年勤務）が年収500万円も低すぎる
- 年商規模と釣り合わない

### 修正後（妥当な設定）

#### 黒澤社長
**設定時給**: **5,000円**（控えめな見積もり）

**根拠**:
- 年収1,000万円相当
- 従業員20-30名規模の経営者として控えめな設定
- 実際はもっと高い可能性もあるが、ROI試算では控えめに

**考慮要素**:
- 最終意思決定者としての責任
- 経営戦略立案の価値
- 「社長の時間を奪う」機会費用

#### 亜希さん
**設定時給**: **3,000円**（ミニマムな見積もり）

**根拠**:
- 年収600万円相当
- 10年勤務、実務責任者、幅広い業務担当
- 黒澤社長の配偶者という特殊な立場も考慮

**考慮要素**:
- 実務のキーパーソン
- 「亜希さんの時間を奪う」機会費用

## 末武の時給設定

### AI講座実績からの逆算
**実績**:
- 金額: 109,091円（2名受講）
- 期間: 1ヶ月（4日間）
- 準備時間も含めると: 実質8-10日間

**逆算**:
- 109,091円 ÷ 10日 = 10,909円/日
- 10,909円 ÷ 8時間 = **約1,364円/時**

**問題点**:
- これは「講義」の対価（教育サービス）
- 準備時間、資料作成、伴走サポート込み
- 単純な時給換算は適切ではない

### 本提案（PDCA組織変革）の時給設定

**提案内容**:
- 基本パッケージ: 18万円 / 3ヶ月
- 使用日数: 約10日間（フル稼働換算）

**逆算**:
- 180,000円 ÷ 10日 = 18,000円/日
- 18,000円 ÷ 8時間 = **2,250円/時**

**問題点**:
- 技術者・コンサルタントとして低すぎる
- 黒澤社長（5,000円/時）の半額以下
- 亜希さん（3,000円/時）より安い
- これでは整合性がない

### 妥当な時給設定（末武）

**フリーランスエンジニア・コンサルタントの相場**:
- 初級（経験1-3年）: 3,000-5,000円/時
- 中級（経験3-7年）: 5,000-8,000円/時
- 上級（経験7年以上）: 8,000-15,000円/時
- 専門家（特定領域）: 10,000-20,000円/時

**末武の場合**:
- AI活用、議事録システム構築、組織変革支援の経験あり
- 建設業界特化の知見
- 13ヶ月の伴走実績

**設定時給**: **4,000-5,000円/時**

**根拠**:
- 中級エンジニア・コンサルタントとして
- 黒澤社長（5,000円）と同等か若干下
- 亜希さん（3,000円）より上
- これで整合性が取れる

## 末武の価格設定の考え方

### AI講座の価格設定（実績）
**金額**: 109,091円（2名、1ヶ月、4日間）

**価格構成**（推定）:
- 講義準備: 3日間 × 20,000円 = 60,000円
- 講義実施: 4日間 × 20,000円 = 80,000円
- 資料作成: 2日間 × 20,000円 = 40,000円
- 伴走サポート: 1日間 × 20,000円 = 20,000円
- **合計**: 10日間 × 20,000円 = 200,000円
- **実際**: 109,091円（約半額）

**理由**（推測）:
- 初めての講座で実績作りのため
- 関係性構築のため
- 値引きした可能性

### 本提案の価格設定（修正案）

#### 現在の設定（問題あり）
- 基本パッケージ: 18万円 / 10日間 = 18,000円/日
- 日給18,000円 = 時給2,250円
- これは経営層（時給3,000-5,000円）より安い

#### 修正案A: 時給4,000円ベース
- 時給4,000円 × 8時間 = 日給32,000円
- 10日間 × 32,000円 = **32万円**
- フルパッケージ（15日間）: **48万円**

#### 修正案B: 時給5,000円ベース
- 時給5,000円 × 8時間 = 日給40,000円
- 10日間 × 40,000円 = **40万円**
- フルパッケージ（15日間）: **60万円**

#### 修正案C: 現実的な折衷案
**考え方**:
- 時給換算ではなく、「プロジェクト単価」として設定
- 構築（高度な技術）: 日給30,000-40,000円
- 伴走サポート（低度な技術）: 日給15,000-20,000円
- 混合で平均25,000円/日

**価格設定**:
- 基本パッケージ（10日間）: **25万円**
- フルパッケージ（15日間）: **37.5万円**

### 価格改定の理由

1. **時給の整合性**: 経営層より安い価格設定は不自然
2. **相場との整合性**: 技術者・コンサルタントとして適正価格
3. **ROIの改善**: 経営層の時給を適正化することで、ROIも改善
4. **透明性**: 時給・日給の根拠を明示
