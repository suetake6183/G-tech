# 固定費用と成果報酬率のバランス設計

**作成日**: 2025年11月14日
**目的**: 広告運用サポートの料金体系設計

---

## 🎯 基本的な考え方

### 原則

```
固定費用が高い → リスク小 → 成果報酬率は低め
固定費用が低い → リスク大 → 成果報酬率は高め

固定費 + 成果報酬の合計 ≒ 「生み出した価値」に対して妥当な額
```

### 工数の概算

末武さんの想定工数：
- 初回診断：1-2時間
- 設定改善作業：5-10時間（タグ設定、キーワード設定、CV設定）
- モニタリング・微調整（3ヶ月）：月2-3時間 × 3ヶ月 = 6-9時間
- **合計：12-21時間程度**

末武さんの時給（参考）：
- AI講座から逆算：2-3万円/時間程度
- **総工数の価値：約24-63万円**

---

## 💰 3つのパターン設計

### パターンA: 完全成果報酬型（推奨）

```
【固定費】
初期費用: 0円
月額費用: 0円

【成果報酬】
報酬率: 生み出した価値の 35-40%

【リスク配分】
クライアント: リスクほぼゼロ（成果なし=支払いゼロ）
末武さん: リスク最大（成果なし=報酬ゼロ、工数すべて無駄）
```

**シミュレーション**（生み出した価値が150万円の場合）：
```
固定費: 0円
成果報酬: 150万円 × 37.5% = 56.25万円
━━━━━━━━━━━━━━━━━━━━
総額: 56.25万円
```

**メリット**:
- ✅ クライアントのリスクゼロ
- ✅ 契約ハードルが最も低い
- ✅ 黒澤工務店の意思決定パターン（リスク回避型）に最適

**デメリット**:
- ❌ 末武さんのリスクが最大
- ❌ 成果が出なければ工数すべてが無駄

---

### パターンB: 少額着手金 + 成果報酬

```
【固定費】
初期費用（着手金）: 5-8万円
　└ 現状診断・設定改善作業の最低限の対価

【成果報酬】
報酬率: 生み出した価値の 25-30%

【リスク配分】
クライアント: 初期投資あり、成果報酬は控えめ
末武さん: 工数の一部は回収済み、リスク中程度
```

**シミュレーション**（生み出した価値が150万円の場合）：
```
固定費: 6.5万円
成果報酬: 150万円 × 27.5% = 41.25万円
━━━━━━━━━━━━━━━━━━━━
総額: 47.75万円
```

**メリット**:
- ✅ 工数の一部は確実に回収
- ✅ クライアントのリスクも低い（着手金のみ）
- ✅ バランスが良い

**デメリット**:
- ❌ 初期投資が必要（予算確保）
- ❌ 「成果報酬型」の訴求力がやや弱まる

---

### パターンC: 月額固定費 + 成果報酬

```
【固定費】
月額費用: 3-5万円/月 × 3ヶ月 = 9-15万円
　└ 継続的な運用管理・モニタリングの対価

【成果報酬】
報酬率: 生み出した価値の 15-20%

【リスク配分】
クライアント: 毎月の支払いあり、成果報酬は低め
末武さん: 工数は継続的に回収、リスク小
```

**シミュレーション**（生み出した価値が150万円の場合）：
```
固定費: 12万円（4万円/月 × 3ヶ月）
成果報酬: 150万円 × 17.5% = 26.25万円
━━━━━━━━━━━━━━━━━━━━
総額: 38.25万円
```

**メリット**:
- ✅ 末武さんのリスクが最小
- ✅ 工数は毎月確実に回収
- ✅ 総額は最も抑えられる

**デメリット**:
- ❌ クライアントの初期負担が大きい
- ❌ 「成果報酬型」の訴求力が弱い
- ❌ 黒澤工務店の意思決定パターンに合わない可能性

## 🎯 推奨案

### 提案方針

**複数の選択肢を提示し、クライアントに選んでもらう**

提案書では、3つのパターンを並列で提示し、それぞれのメリット・デメリットを説明。ただし、**パターンAを推奨**として明示する。

### パターンAを推奨する理由

1. **黒澤工務店の意思決定パターンに最適**
   - リスク回避型
   - 段階的アプローチを好む
   - 初期投資への抵抗が強い可能性

2. **「成果報酬型」の訴求力が最大**
   - 「成果が出なければ費用ゼロ」は強いメッセージ
   - 信頼関係構築の第一歩として最適

3. **末武さんの誠実な姿勢と一貫性**
   - 「絶対できるとは言えない」→ 成果が出たら業界標準の報酬
   - 論理的に一貫している

### ただし、柔軟性も確保

無料診断後、クライアントの状況や予算感に応じて：
- 「パターンBの方が安心ですか？」と提案
- 「継続的なサポートが欲しい場合はパターンC」も選択肢

## 💡 料金設定の透明性確保

### 提案書での説明案

```markdown
## 料金体系（3つの選択肢）

### パターンA: 完全成果報酬型（推奨）

**固定費**: 0円
**成果報酬**: 生み出した価値の 35-40%

【こんな方におすすめ】
- 初期投資を抑えたい
- 成果が出るか不安なので、リスクゼロで始めたい
- まずは試してみたい

【試算例】
3ヶ月で月間コスト削減5万円 × 6ヶ月持続 = 150万円の価値
→ 成果報酬: 150万円 × 37.5% = 56.25万円

━━━━━━━━━━━━━━━━━━━━

### パターンB: 少額着手金 + 成果報酬

**固定費**: 5-8万円（初回のみ）
**成果報酬**: 生み出した価値の 25-30%

【こんな方におすすめ】
- 少額の初期投資はOK
- 成果報酬を抑えたい
- バランス重視

【試算例】
着手金: 6.5万円
成果報酬: 150万円 × 27.5% = 41.25万円
総額: 47.75万円

━━━━━━━━━━━━━━━━━━━━

### パターンC: 月額固定費 + 成果報酬

**固定費**: 3-5万円/月 × 3ヶ月
**成果報酬**: 生み出した価値の 15-20%

【こんな方におすすめ】
- 継続的なサポートが欲しい
- 毎月の支払いは問題ない
- 成果報酬を最小限に抑えたい

【試算例】
月額固定費: 4万円 × 3ヶ月 = 12万円
成果報酬: 150万円 × 17.5% = 26.25万円
総額: 38.25万円
```

## 📝 料金設定の根拠説明

提案書に以下の説明を追加：

```markdown
### なぜこの料金設定なのか？

**業界標準との比較**:
- 成果報酬型コンサル（コストダウン、営業代行等）: 価値の30-50%が相場
- 私たちの提案: パターンAで35-40%、パターンB・Cはさらに控えめ

**固定費と成果報酬のバランス**:
- 固定費が少ない = リスクを私たちが負う = 成果報酬は高め
- 固定費が多い = リスクが分散 = 成果報酬は控えめ

どのパターンも、最終的な総額は「生み出した価値」に対して
適正な範囲になるよう設計しています。

無料診断後、御社の状況や予算感に合わせて、
一緒に最適なパターンを選びましょう。
```

---

## 結論

**パターンA（完全成果報酬型、35-40%）を推奨**として提示しつつ、パターンB・Cも選択肢として提示する。

この設計により：
- ✅ クライアントのリスクを最小化
- ✅ 柔軟性を確保（予算感に応じて選択可能）
- ✅ 透明性が高い（なぜこの料金なのかが明確）
- ✅ 末武さんのリスクも適切に報酬に反映
