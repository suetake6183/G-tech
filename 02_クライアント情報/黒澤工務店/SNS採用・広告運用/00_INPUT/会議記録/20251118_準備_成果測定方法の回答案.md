# 成果測定方法の回答案

**作成日**: 2025年11月17日
**対象**: 亜希さんからの質問「報酬について昨対月と比較しますか？」への回答

---

## 🎯 結論

**推奨する測定方法**: 「改善前3ヶ月 vs 改善後3ヶ月」の比較

**理由**: 広告改善の「純粋な効果」を正確に測定するため

---

## 📊 3つの比較方法の違い

### 方法①: 昨対月比較（前年同月比）❌ 非推奨

```
2024年11月のCPA vs 2025年11月のCPA
```

**問題点**:
- 季節変動の影響を受ける
  - 例: 建築業界は春・秋が繁忙期、夏・冬は閑散期
  - 例: 採用は3-4月、9-10月が活発
- 他の要因（市況、競合、自社施策）の影響を分離できない
- 広告改善の「純粋な効果」が測定できない

**具体例**:
```
2024年11月: CPA 3万円（繁忙期）
2025年11月: CPA 2.5万円（繁忙期）

→ 改善したように見えるが、広告改善の効果なのか、
  市況の変化なのか、判別できない
```

---

### 方法②: 前月比較（直近1ヶ月 vs 翌1ヶ月）△ やや不適切

```
改善前1ヶ月 vs 改善後1ヶ月
```

**問題点**:
- データのブレが大きい（1ヶ月だけでは統計的に不安定）
- 季節変動の影響を受けやすい
- たまたま良い/悪い月を比較してしまうリスク

**具体例**:
```
10月: CPA 3万円（たまたま好調な月）
11月: CPA 2.5万円（改善実施後）

→ 改善効果なのか、たまたまなのか判別困難
```

---

### 方法③: 改善前後比較（3ヶ月平均）✅ 推奨

```
改善前3ヶ月の平均 vs 改善後3ヶ月の平均
```

**メリット**:
- データのブレを平均化できる（統計的に安定）
- 季節変動の影響を最小化
- 広告改善の「純粋な効果」を測定できる
- **ベースライン測定**の考え方に基づく

**具体例**:
```
【改善前3ヶ月（8月〜10月）の平均】
CPA: 3万円
月間CV数: 10件
月間広告費: 30万円

【改善後3ヶ月（11月〜1月）の平均】
CPA: 2万円（改善率: 33%）
月間CV数: 15件（増加率: 50%）
月間広告費: 30万円（同じ）

→ 明確に「改善の効果」を測定できる
```

---

## 💡 ベースライン測定とは？

### 定義

**ベースライン**: 広告改善を行わなかった場合に期待される数値

**ベースライン測定**: キャンペーン（改善）の前後でデータを比較し、純粋な効果を測定する手法

### 実務での使われ方

デジタル広告業界では、広告改善の効果を測定する際の標準的な手法として用いられています。

**具体的な算出方法**:
1. 改善前の一定期間（通常3ヶ月）の平均値を「ベースライン」とする
2. 改善後の同じ期間の平均値を測定
3. 差分が「改善の効果」

---

## 📐 具体的な測定プロセス

### ステップ1: ベースライン期間の設定

**改善前3ヶ月**（例: 2025年8月〜10月）
- この期間の平均値を「ベースライン」とする
- Google広告とYahoo広告の両方のデータを集計

**記録すべき指標**:
- 月間広告費
- 月間インプレッション数
- 月間クリック数、CTR
- 月間コンバージョン数、CVR
- CPA

---

### ステップ2: 改善実施

**2025年11月**
- タグ設定改善
- キーワード設定改善
- コンバージョン設定改善
- マイクロコンバージョン設定

---

### ステップ3: 効果測定期間

**改善後3ヶ月**（例: 2025年11月〜2026年1月）
- 改善後の効果が安定するまで待つ（約3ヶ月）
- 同じ指標を記録

**なぜ3ヶ月必要か**:
- 広告の学習期間（アルゴリズムの最適化）
- データのブレを平均化
- 季節変動の影響を最小化

---

### ステップ4: 比較・評価

```
【CPA改善率】
改善前平均CPA: 3万円
改善後平均CPA: 2万円
→ 改善率: 33%

【CV数増加率】
改善前平均CV数: 10件/月
改善後平均CV数: 15件/月
→ 増加率: 50%

【生み出した価値の算出】
CV 1件の価値: 30万円（仮定）
月間増加CV数: 5件
月間の価値向上: 150万円
持続期間: 6ヶ月（設定改善の効果持続）
→ 総価値: 900万円
```

## 🎯 亜希さんへの回答案

### 質問: 「報酬について昨対月と比較しますか？」

**回答**:

昨対月（前年同月比）ではなく、**「改善前3ヶ月の平均 vs 改善後3ヶ月の平均」**で比較することをご提案します。

**理由は3つあります**:

1. **季節変動の影響を排除できる**
   - 昨対月だと、繁忙期・閑散期の影響を受けてしまいます
   - 建築業界は季節で大きく変動するため、正確な測定が困難です

2. **広告改善の純粋な効果を測定できる**
   - 「ベースライン測定」という業界標準の手法です
   - 改善前後の同じ期間を比較することで、改善の効果だけを抽出できます

3. **データが安定する**
   - 3ヶ月の平均を取ることで、たまたま良い/悪い月の影響を排除
   - 統計的に信頼性の高い測定が可能

**具体的な測定方法**:

```
【ベースライン期間】
改善前3ヶ月（例: 8月〜10月）の平均値を記録

【改善実施】
11月に設定改善を実施

【評価期間】
改善後3ヶ月（11月〜1月）の平均値を記録

【成果の判定】
CPA改善率、CV数増加率を算出
→ 生み出した価値を金額換算
```

このように測定することで、御社にとっても私にとっても、公平で透明性の高い評価が可能になります。

## 💬 想定される質問と回答

### Q1: なぜ3ヶ月も必要なのですか？

**A**: 3つの理由があります。

1. 広告の学習期間（アルゴリズムが最適化されるまで約1ヶ月）
2. データのブレを平均化（1ヶ月だけでは偶然の良し悪しがある）
3. 季節変動の影響を最小化（3ヶ月あれば季節の偏りを緩和）

### Q2: もっと早く成果を確認できませんか？

**A**: 途中経過は毎月確認できます。

ただし、最終的な評価（成果報酬の算定）は3ヶ月後がフェアです。
例えば、1ヶ月目に偶然良い数字が出ても、2ヶ月目に戻ることがあります。
3ヶ月の平均を見ることで、より正確な評価ができます。

### Q3: 3ヶ月後に成果が出なかったらどうなりますか？

**A**: 完全成果報酬型（パターンA）の場合、費用は一切発生しません。

ただし、途中経過で「このままでは成果が出そうにない」と判明した場合は、
その時点で相談させてください。追加の改善策を提案するか、
契約を終了するかを一緒に判断しましょう。
