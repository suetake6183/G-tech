# 業務改善 事前アンケート Googleフォーム設計書

**作成日**: 2025年12月1日
**目的**: 業務課題の洗い出しと改善優先順位の決定

---

## フォーム基本設定

- **フォームタイトル**: 業務改善 事前アンケート
- **説明文**:
  ```
  業務改善を進めるにあたり、皆さんの日々の業務についてお聞かせください。
  率直なご意見をいただけると、より効果的な改善につなげることができます。
  所要時間は5〜10分程度です。
  ```
- **回答の収集設定**: メールアドレスを収集しない（匿名可）
- **進行状況バー**: 表示する

---

## セクション1: 基本情報

### Q1. お名前（任意）
- **形式**: 記述式（短文）
- **必須**: いいえ
- **説明**: 匿名での回答も可能です

### Q2. 主な担当業務
- **形式**: チェックボックス（複数選択可）
- **必須**: はい
- **選択肢**:
  - 現場作業（施工）
  - 現場管理
  - 見積作成
  - 図面作成
  - 顧客対応
  - 事務作業（書類作成・データ入力など）
  - 経理・請求業務
  - 広告・集客関連
  - その他（自由記述）

## セクション2: 時間がかかっている業務

**セクション説明**:
```
日々の業務の中で、時間がかかっていると感じるものについてお聞きします。
```

### Q3. 時間がかかっていると感じる業務を教えてください
- **形式**: チェックボックス（複数選択可）
- **必須**: はい
- **選択肢**:
  - 書類作成（見積書、請求書、報告書など）
  - 情報を探す（過去の資料、顧客情報など）
  - データ入力・転記
  - 社内での確認・承認待ち
  - メール・電話対応
  - 会議・打ち合わせ
  - 移動時間
  - 写真の整理・管理
  - スケジュール調整
  - その他（自由記述）

### Q4. 最も時間短縮したい業務を1つ選んでください
- **形式**: 記述式（短文）
- **必須**: はい
- **説明**: Q3で選んだ中から、特に改善したいものを1つ書いてください

### Q5. その業務にかかる時間を教えてください
- **形式**: ラジオボタン（単一選択）
- **必須**: はい
- **選択肢**:
  - 1回あたり15分未満
  - 1回あたり15分〜30分
  - 1回あたり30分〜1時間
  - 1回あたり1時間〜2時間
  - 1回あたり2時間以上

### Q6. その業務の頻度を教えてください
- **形式**: ラジオボタン（単一選択）
- **必須**: はい
- **選択肢**:
  - 毎日
  - 週に2〜3回
  - 週に1回
  - 月に2〜3回
  - 月に1回以下

## フォーム作成手順

### Googleフォームでの作成方法

1. Google Driveで「新規」→「Googleフォーム」→「空白のフォーム」
2. フォームタイトルと説明文を入力
3. 上記のセクションと質問を順番に追加
4. 各セクションの区切りは「セクションを追加」で作成
5. 設定タブで以下を確認：
   - 「メールアドレスを収集する」→ オフ
   - 「回答を1回に制限する」→ オフ（匿名回答のため）
   - 「進行状況バーを表示」→ オン

### 回答後の分析ポイント

- Q4〜Q6の組み合わせで「週あたりの総負担時間」を算出
- Q7で負担の原因を分類し、解決策の方向性を決定
- Q8〜Q9でリスクの高い業務を特定
- Q11で改善効果の期待値を把握
- Q12でIT施策の難易度を判断

## 経営者向け追加ヒアリング項目（別途実施推奨）

社員アンケートとは別に、経営者には以下を確認することを推奨します。

1. **会社として最優先で解決したい課題は何か**
2. **投資してでも改善したい業務はあるか**
3. **社員が気づいていないが、経営視点で問題だと感じていること**
4. **改善に対する社員の協力体制について懸念はあるか**
5. **過去に業務改善を試みて失敗した経験はあるか**

これらは対面またはオンラインMTGで確認し、社員アンケート結果と照らし合わせることで、優先順位の最終決定に活用します。
