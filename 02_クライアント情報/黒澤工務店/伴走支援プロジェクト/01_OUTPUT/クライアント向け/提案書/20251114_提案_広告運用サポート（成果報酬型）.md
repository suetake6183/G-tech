## 📋 はじめに

この度は、Google広告・Yahoo広告の運用サポートについてご相談いただき、ありがとうございます。

### ご相談内容の確認

**Google広告**
- 各広告のタグ設定（マイクロコンバージョンとして計測したいトリガーとの紐づけ）
- 添付画像に記載の、設定が不適切・要確認の箇所への対応

**Yahoo広告**
- ディスプレイ広告のキーワード設定
- コンバージョン設定

また、「そもそも取り組めていない部分がありそう」とのことで、御社の広告を拝見しながらのご教授も可能かとのご質問もいただきました。

### サポート可能性について

結論：**サポートできる可能性は高い**と考えています。

ただ、正直に申し上げて広告運用は私の専門分野ではありません。
広告出稿の経験はあり、不足している知識はリサーチ等で補完可能と想定しています。

そのため、**完全成果報酬型**でのサポートをご提案します。
成果が出なければ費用は一切いただきません。

## 🎯 提案の進め方（3ステップ）

いきなり契約や報酬額を決めるのではなく、段階的に進めることをご提案します。

```
ステップ1: 無料ヒアリング・現状診断
         ↓
ステップ2: 費用対効果の試算
         ↓
ステップ3: 料金体系の選択
         ↓
     契約判断
```

### なぜこの進め方なのか？

**御社のリスクをゼロにするため**
まず無料で現状を診断し、改善可能性を確認します。
費用対効果が見込めない場合は、その時点で終了できます。

**透明性を確保するため**
「どんな成果が出れば、どれくらいの価値があるか」を一緒に考えます。
報酬額も一方的に決めるのではなく、費用対効果に基づいて設定します。

**納得感を持って判断いただくため**
数字に基づいた客観的な判断材料を提供します。
「やってみないとわからない」ではなく、「これくらいの改善が期待できそう」という見通しを立てます。

## 📊 ステップ1: 無料ヒアリング・現状診断

### ヒアリング内容

以下の情報を教えていただきます。

**広告の基本情報**
- 広告の目的（採用？受注獲得？ブランディング？）
- 現在の予算（月額いくら使っているか）
- 運用期間（いつから運用しているか）
- 運用体制（誰が設定・管理しているか）

**現在のパフォーマンス**
- 月間インプレッション数（広告が何回表示されているか）
- 月間クリック数・CTR（何回クリックされているか、クリック率）
- 月間コンバージョン数・CVR（何件の成果があるか、コンバージョン率）
- 現在のCPA（1件の成果を得るのにかかっている費用）

**これまでの経緯**
- なぜ行き詰まったか
- 期待する成果（どうなったら嬉しいか）

### 診断内容

ヒアリング内容を元に、以下を診断します。

1. **現状の問題点の特定**
タグ設定、キーワード設定、コンバージョン設定の不備を洗い出します。

2. **改善案の提示**
具体的にどこをどう直すか、優先順位をつけて説明します。

3. **改善効果の予測**
どの程度の改善が期待できそうか、控えめな予測と楽観的な予測の幅を提示します。

### 実施方法・所要時間

- **実施方法**: オンライン会議（Zoom等）または対面
- **所要時間**: 1〜2時間程度
- **費用**: **完全無料**

## 💰 ステップ2: 費用対効果の試算

診断結果を元に、御社にとっての費用対効果を試算します。

### 試算の流れ

#### 1. 改善シミュレーション

例えば、現状が以下の場合：
```
月間広告費: 30万円
月間CV数: 10件
現在のCPA: 3万円/件
```

改善後の予測：
```
【控えめな予測】
月間CV数: 15件（+5件）
改善後のCPA: 2万円/件（-1万円）
→ 月間で5万円のコスト削減 + CV数増加

【楽観的な予測】
月間CV数: 20件（+10件）
改善後のCPA: 1.5万円/件（-1.5万円）
→ 月間で15万円のコスト削減 + CV数増加
```

#### 2. 御社にとっての価値算出

CVの価値を金額換算します。

**採用広告の場合**
```
1件の応募 → 面接 → 採用の流れ
採用1名あたりの価値: 50万円（他の採用手段との比較）
応募から採用までの確率: 20%

→ 応募1件の価値 = 10万円
→ 月間CV +5件 = 50万円の価値
```

**受注獲得広告の場合**
```
1件の問い合わせ → 商談 → 受注の流れ
受注1件あたりの粗利: 100万円
問い合わせから受注までの確率: 30%

→ 問い合わせ1件の価値 = 30万円
→ 月間CV +5件 = 150万円の価値
```

#### 3. ROI（投資対効果）の算出

```
【控えめな予測の場合】
月間の価値向上: 50万円
広告改善の持続期間: 6ヶ月（設定改善の効果持続）
→ 総価値: 300万円

【楽観的な予測の場合】
月間の価値向上: 150万円
広告改善の持続期間: 6ヶ月
→ 総価値: 900万円
```

**この価値に対して、いくらの報酬が妥当か？**を一緒に考えます。

## 🤝 ステップ3: 料金体系の選択

費用対効果の試算を元に、3つの料金体系から選んでいただきます。

### 基本的な考え方

```
固定費用が少ない = 御社の初期負担が小さい = 成果報酬は高め
固定費用が多い = 御社の初期負担あり = 成果報酬は控えめ
```

業界標準では、成果報酬型コンサルティング（コストダウン、営業代行等）の報酬率は**「生み出した価値の30〜50%」**が相場です。

今回は、固定費用とのバランスを考慮し、3つのパターンをご用意しました。

---

### パターンA: 完全成果報酬型（推奨）

```
【固定費】
初期費用: 0円
月額費用: 0円

【成果報酬】
報酬率: 生み出した価値の 35〜40%

【評価期間】
3ヶ月（広告改善の効果が安定するまで）
```

**こんな場合におすすめ**
- 初期投資を抑えたい
- 成果が出るか不安なので、リスクゼロで始めたい
- まずは試してみたい

**試算例**（控えめな予測の場合）
```
3ヶ月で月間コスト削減5万円 × 6ヶ月持続 = 150万円の価値
→ 成果報酬: 150万円 × 37.5% = 56.25万円
```

---

### パターンB: 少額着手金 + 成果報酬

```
【固定費】
初期費用（着手金）: 5〜8万円
└ 現状診断・設定改善作業の最低限の対価

【成果報酬】
報酬率: 生み出した価値の 25〜30%

【評価期間】
3ヶ月
```

**こんな場合におすすめ**
- 少額の初期投資はOK
- 成果報酬を抑えたい
- バランス重視

**試算例**（控えめな予測の場合）
```
着手金: 6.5万円
成果報酬: 150万円 × 27.5% = 41.25万円
━━━━━━━━━━━━━━━━━━━━
総額: 47.75万円
```


---

### パターンC: 月額固定費 + 成果報酬

```
【固定費】
月額費用: 3〜5万円/月 × 3ヶ月 = 9〜15万円
└ 継続的な運用管理・モニタリングの対価

【成果報酬】
報酬率: 生み出した価値の 15〜20%

【評価期間】
3ヶ月
```

**こんな場合におすすめ**
- 継続的なサポートが欲しい
- 毎月の支払いは問題ない
- 成果報酬を最小限に抑えたい

**試算例**（控えめな予測の場合）
```
月額固定費: 4万円 × 3ヶ月 = 12万円
成果報酬: 150万円 × 17.5% = 26.25万円
━━━━━━━━━━━━━━━━━━━━
総額: 38.25万円
```

### 3パターンの比較表

| 項目 | パターンA | パターンB | パターンC |
|------|-----------|-----------|-----------|
| **固定費** | 0円 | 5〜8万円 | 9〜15万円 |
| **成果報酬率** | 35〜40% | 25〜30% | 15〜20% |
| **総額**（150万円の価値の場合） | 56.25万円 | 47.75万円 | 38.25万円 |
| **御社のリスク** | ゼロ | 小 | 中 |
| **契約ハードル** | 最も低い | 中 | 高い |

### 私の推奨

**パターンA（完全成果報酬型）**を推奨します。

**理由**
- 御社のリスクが完全にゼロ
- 「成果が出なければ費用ゼロ」は最も分かりやすい
- まずは試していただき、効果を実感してから継続を検討できる

ただし、御社の予算感やご希望に応じて、パターンB・Cも選択可能です。
診断後に、一緒に最適なパターンを決めましょう。

## 💡 なぜこの報酬率なのか？

### 成果報酬率の考え方

**完全成果報酬型（パターンA）で報酬率35〜40%の理由**

**1. 御社のリスクが完全にゼロ**

完全成果報酬型 = 成果が出なければ費用は一切発生しません。
御社は初期投資ゼロ、ランニングコストもゼロで始められます。

**2. 成果が出た時点で「価値を生み出した」**

経験の有無に関わらず、結果として御社に価値をもたらしたということです。
だから、報酬率は業界標準（30〜50%）が妥当だと考えています。

**3. 業界標準との比較**
- コストダウンコンサル: 削減額の30〜50%
- 営業代行: 売上額の30〜50%
- 採用代行: 年収の30〜35%
- **私たちの提案**: 35〜40%（業界標準の範囲内）

### 固定費と成果報酬のバランス

- **パターンA**: 固定費0円 → 御社の初期負担なし → 成果報酬は高め（35〜40%）
- **パターンB**: 固定費あり → 御社の初期負担あり → 成果報酬は中程度（25〜30%）
- **パターンC**: 固定費大 → 御社の月次負担あり → 成果報酬は低め（15〜20%）

どのパターンも、最終的な総額は「生み出した価値」に対して適正な範囲になるよう設計しています。

## 📝 成果の測定方法

### 成果の定義

診断後、御社が重視する指標を一緒に決めます。

**選択肢1: CPA（顧客獲得単価）改善率**
```
例: CPA 30%改善 → 成果達成
```

**選択肢2: コンバージョン数増加**
```
例: 月間CV +10件 → 成果達成
```

**選択肢3: ROAS（広告費用対効果）改善**
```
例: ROAS 200%→300% → 成果達成
```

### 評価タイミング

- **評価期間**: 3ヶ月後
- **測定方法**: Google広告・Yahoo広告の管理画面データ
- **比較基準**: 改善前3ヶ月の平均 vs 改善後3ヶ月の平均

## 💬 よくある質問

### Q1: 本当に成果が出るか不安です

正直に申し上げて、絶対に成果が出るとは言い切れません。
だからこそ、完全成果報酬型（パターンA）をご提案しています。

成果が出なければ、御社の費用負担は一切ありません。
まずは診断を受けていただき、改善可能性を確認してから判断いただけます。

### Q2: 専門分野ではないとのことですが、大丈夫ですか？

広告出稿の経験はありますし、設定改善（タグ設定、キーワード設定、CV設定）は技術的に対応可能です。

ただし、不足している知識はリサーチで補完しますので、多少の時間をいただく場合があります。
御社に安心していただくため、成果が出た場合のみ報酬をいただく仕組みにしています。

### Q3: 成果報酬率35〜40%は高くないですか？

一見高く感じるかもしれませんが、業界標準（30〜50%）の範囲内です。

また、完全成果報酬型なので、成果が出なければ費用はゼロです。
もし初期投資が可能であれば、パターンB（報酬率25〜30%）やパターンC（報酬率15〜20%）も選択できます。

### Q4: 継続的なサポートは可能ですか？

はい、可能です。

3ヶ月の成果報酬型サポート後、継続を希望される場合は、新しい料金体系（月額固定費 + 成果報酬など）を検討できます。

### Q5: クリエイティブ制作（バナー、広告文）もお願いできますか？

今回のご提案は「設定改善」が中心です。

クリエイティブ制作は含まれていませんが、必要であれば別途ご相談ください。

### Q6: 途中で「やっぱりやめたい」となった場合は？

無料ヒアリング・診断後、費用対効果の試算を見て「やめる」判断をされても全く問題ありません。

契約前なので費用は一切発生しません。

### Q7: パターンAとパターンBで迷っています

診断後、改善の難易度や期待効果が明確になってから決めていただいて構いません。

その時点で、一緒に最適なパターンを検討しましょう。
