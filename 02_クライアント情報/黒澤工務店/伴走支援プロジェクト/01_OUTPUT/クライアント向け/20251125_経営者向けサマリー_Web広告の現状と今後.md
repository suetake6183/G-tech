## 広告効果をさらに上げるための改善案

現在のデータを分析した結果、**追加費用なしで成果を10〜20%向上**できる可能性があります。

### 改善案1: 広告を出す時間帯を変える（費用: ¥0）

現在、深夜〜早朝にも広告が配信されていますが、この時間帯は効率が悪いことがわかりました。

```
┌─────────────────────────────────────────────────────────────┐
│  時間帯別の広告効率                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  20〜23時  ████████████████████  最も反応が良い（効率◎）   │
│  17〜19時  ████████████████      反応が良い（効率○）       │
│  12〜14時  ████████████          まあまあ（効率○）         │
│  0〜6時    ████                  反応が少ない（効率×）     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**やること**: 深夜0〜6時の広告配信を止めて、夜20〜23時に予算を集中させる

**期待効果**: 同じ予算で **お問い合わせ5〜10件増加**

---

### 改善案2: 効率の良いキャンペーンに予算を移す（費用: ¥0）

現在3つのキャンペーンを運用していますが、効率に差があります。

| キャンペーン | 使った費用 | 獲得件数 | 1件あたりのコスト | 評価 |
|-------------|-----------|---------|------------------|------|
| 常時OPENHOUSE | ¥48,669 | 7件 | **¥6,953** | 効率◎ |
| コンバージョン | ¥300,375 | 38件 | ¥7,905 | 効率○ |
| トラフィック | ¥30,226 | 認知向け | - | 認知◎ |

**やること**: 「常時OPENHOUSE」の予算を増やし、「コンバージョン」から一部移す

**期待効果**: 同じ予算で **お問い合わせ5〜10件増加**

---

### 改善案3: サイトを訪問した人への再アプローチ（費用: ¥0〜少額）

一度サイトに来たけど問い合わせしなかった人に、再度広告を見せる方法があります。

特に効果的なのは：
- **施工事例ページを見た人**（関心が高い）
- **お問い合わせページまで来たけど送信しなかった人**（あと一歩だった人）

**やること**: これらの人だけに絞った広告を追加で設定

**期待効果**: **追加で月5〜10件のお問い合わせ獲得**

---

### 改善案4: 動画広告の追加（費用: 制作費が必要）

現在は静止画の広告ですが、動画広告は反応が **2〜3倍** 良い傾向があります。

**効果的な動画の例**:
- 施工事例のルームツアー（15〜30秒）
- お客様の声インタビュー
- モデルハウスの紹介

**期待効果**: 同じ広告費で **リーチ数1.5〜2倍**

---

### 改善案5: ホームページの改善（費用: 制作費が必要）

現在、広告からサイトに来た人の **約6割がすぐ離脱** しています。
これを改善すれば、広告の効果が大きく上がります。

**やること**:
- 広告専用のページを作成
- 「見学予約」「資料請求」ボタンを目立つ位置に
- スマホで見やすいデザインに（閲覧者の66%がスマホ）

**期待効果**: 同じ広告費で **お問い合わせ数1.5〜2倍**

## 参考：用語の説明

| 用語              | 意味                         |
| --------------- | -------------------------- |
| **リーチ**         | 広告を見た人の数（同じ人が何度見ても1人とカウント） |
| **コンバージョン（CV）** | お問い合わせや予約など、目標とする行動をした数    |
| **CPA**         | 1件のお問い合わせを獲得するのにかかった費用     |
| **CTR**         | 広告を見た人のうち、クリックした人の割合       |
| **CPC**         | 1クリックあたりの費用                |
| **オーガニック検索**    | 広告ではない、通常のGoogle検索からの流入    |
| **直帰率**         | サイトに来て、1ページだけ見てすぐ離れた人の割合   |
