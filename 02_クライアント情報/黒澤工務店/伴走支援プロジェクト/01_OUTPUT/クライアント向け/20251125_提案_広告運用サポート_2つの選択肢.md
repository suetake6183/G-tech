## この提案について

先日の広告運用データ分析の結果、追加の広告費をかけずに成果を向上させる余地があることが分かりました。

本提案は、その改善を実行するためのサポート契約についてです。

## 現状の確認

### 広告運用の実績（過去30日間）

| 広告媒体 | 広告費 | クリック数 | CV数 | CPA |
|---------|--------|-----------|------|-----|
| Meta広告（FB/Instagram） | 約38万円 | 1,417回 | 45件 | 約8,400円 |
| Google広告 | 約3.1万円 | 256回 | 測定未設定 | - |
| Yahoo広告 | 約0.9万円 | 41回 | 測定未設定 | - |
| **合計** | **約42万円** | **1,714回** | **45件** | - |

Meta広告は業界平均（1.5〜3万円）の半分以下のCPAであり、効率的に運用できています。

一方、**Google広告・Yahoo広告はコンバージョン測定が未設定**のため、成果が把握できていません。特にGoogle広告はCTR 3.26%（業界平均1-2%）、CPC ¥121と効率が良く、測定設定すれば隠れた成果が見える可能性があります。

### 改善の余地

| 改善施策 | 追加費用 | 期待効果 |
|---------|---------|---------|
| 時間帯の最適化（深夜帯停止） | ¥0 | +5〜10件/月 |
| 予算の再配分（効率の良いキャンペーンへ） | ¥0 | +5〜10件/月 |
| リターゲティング強化 | ¥0〜少額 | +5〜10件/月 |
| Google・Yahoo広告のCV測定設定 | ¥0 | 成果の可視化 |

追加の広告費なしで、月10〜20件の成果向上が見込めます。また、Google・Yahoo広告の測定設定により、全体の広告効果が正確に把握できるようになります。

### なぜ改善できると言えるのか

現状、広告運用において「毎月数字を見て、なぜその結果になったのか仮説を立て、次に何をすべきか考え、実行し、検証する」というサイクルが回っていません。

このPDCAを回すだけで、成果は上がります。

特別なことをするわけではありません。広告運用の基本を、毎月愚直にやり続ける。それだけです。

御社の広告はすでに業界平均の半分以下のCPAで運用できています。ただし、「なぜ良いのか」「どうすればもっと良くなるのか」が把握できていない状態でもあります。ここを明確にし、意図的に改善していけば、さらに良くなる余地は十分にあります。

## 2つの料金プラン

### プランA: 固定月額制

```
月額: 77,000円（税抜）
```

毎月の支払いが固定なので、予算管理がしやすい形です。

成果が出ても出なくても金額は変わりません。私としても成果に関わらず固定で受ける形になるため、そのリスク分を含めた金額設定にしています。

---

### プランB: 成果報酬制（CPA改善ベース）

```
基本料金: 33,000円（税抜）/月
成果報酬: CPAが改善した場合、削減できた広告費の30%
```

「私の改善によって御社が節約できた広告費の30%をいただく」という考え方です。

成果報酬部分については、御社が得をした分の一部をいただく形です。ただし、基本料金33,000円は成果に関わらず発生するため、CPA改善による削減効果が小さい月は、基本料金分がコストとしてかかります。

#### 計算方法

1. 契約開始時点のCPAを「基準CPA」として設定
2. 毎月、実績CPAと基準CPAを比較
3. 改善していた場合、以下の計算で成果報酬を算出

```
削減額 = (基準CPA - 実績CPA) × 今月のコンバージョン数
成果報酬 = 削減額 × 30%
```

#### 計算例

**CPAが改善した場合**

```
基準CPA: 8,400円
今月の実績: 55件獲得、CPA 7,000円

基準CPAで55件取った場合の広告費: 8,400円 × 55件 = 462,000円
実際の広告費: 7,000円 × 55件 = 385,000円
削減額: 77,000円

成果報酬: 77,000円 × 30% = 23,100円
月額合計: 33,000円 + 23,100円 = 56,100円
```

**CPAが改善しなかった場合**

```
基準CPA: 8,400円
今月の実績: 40件獲得、CPA 9,000円

基準CPAより悪化 → 成果報酬: 0円
月額合計: 33,000円のみ
```

## 私の推奨

**プランBをお勧めします。**

理由は以下の通りです。

1. **成果に連動した料金体系**
   - 成果が出なければ基本料金のみ
   - 成果が出た場合、その改善分に応じて報酬が決まる

2. **私のモチベーション**
   - 改善すればするほど報酬が増えるので、成果を出すことに全力で取り組める

3. **Win-Winの構造**
   - 御社が得をしたら、私も得をする

## 次のステップ

本提案の金額や基準CPAは、直近30日間のデータを基にした暫定値です。

正式な契約の前に、以下の流れで進めたいと考えています。

1. **過去1〜2年分の広告運用データを確認する**（私の方で取得します）
   - 季節変動や年間の傾向を把握する

2. **データを確認した上で、基準CPAを一緒に決める**
   - 必要に応じて料金設定も調整

3. **プランを決定し、契約書を作成**

4. **契約後、速やかに改善施策の実施に着手**
