# 黒澤工務店への広告運用サービス設計：最終結論

**作成日**: 2025年11月25日

---

## はじめに：この資料の目的

この資料は、黒澤工務店に対する広告運用サービスをどのように設計すべきかについて、これまでの議論を踏まえた最終結論を示すものである。

結論に至るまでの論理構造を明確にし、なぜそのプランが最適なのかを論証する。

---

## 第1章：前提条件の整理

### 1-1. 黒澤工務店との関係性の全体像

黒澤工務店との関係は、「広告運用」という単一の案件ではない。現在、以下の複数のプロジェクトが同時に進行している。

| プロジェクト | 状況 | 金額規模 |
|-------------|------|---------|
| AI導入・DX推進 | 完了（講座36万円） | 完了済み |
| 情報システム刷新 | 提案済み、11/28決定 | 170〜200万円（11ヶ月） |
| 広告運用サポート | 検討中 | 今回の検討対象 |

つまり、広告運用サービスの設計は、「情報システム刷新（提案C）」との関係を考慮せずには決められない。

### 1-2. 提案Cの内容（情報システム刷新）

提案Cは、以下の内容で設計されている。

**目的**: IT担当を育成し、組織が自走できる状態を作る

**期間・料金**:
- Phase 1（6ヶ月）: 月額21.5〜25万円 → 合計132.5〜155万円
- Phase 2（3ヶ月）: 月額10〜12.5万円 → 合計30〜37.5万円
- Phase 3（2ヶ月）: 月額4〜5万円 → 合計8〜10万円
- **総額**: 170.5〜202.5万円

**含まれる作業**（Phase 1の月43〜50時間）:
- タスク管理システム構築・運用
- 議事録作成プロンプト開発
- 業務洗い出し・分析
- マニュアル作成
- PDCAサイクル確立
- IT担当育成

**含まれないもの**:
- 広告運用

### 1-3. 広告運用の現状

Meta広告の運用状況は以下の通り。

| 項目 | 数値 |
|------|------|
| 月間広告費 | 約39万円 |
| コンバージョン数 | 45件/月 |
| CPA | 約8,400円（業界平均の半分以下） |

**改善余地**:
- 時間帯最適化（深夜帯停止）: +5〜10件/月
- 予算再配分（効率の良いキャンペーンへ）: +5〜10件/月
- リターゲティング強化: +5〜10件/月
- **合計**: +10〜20件/月（追加費用なし）

## 第2章：論点の整理

### 2-1. 検討すべき選択肢

広告運用サービスの設計には、以下の選択肢がある。

**選択肢A**: 提案Cに広告運用を統合する（パッケージ化）
**選択肢B**: 広告運用を別契約として提案Cと並行する
**選択肢C**: 広告運用を単独で提案する（提案Cとは独立）

### 2-2. 各選択肢の検討

#### 選択肢A: 提案Cへの統合

**想定される形**:
- 提案Cの料金に広告運用工数を上乗せ
- または、提案Cの「実務代行」の中に広告運用を含める

**問題点**:

1. **価値の性質が異なる**
   - 提案Cは「プロセス価値」（組織変革、人材育成、仕組み構築）
   - 広告運用は「成果価値」（コンバージョン数、CPA改善）
   - 異なる性質の価値を一つの契約にまとめると、評価基準が曖昧になる

2. **成果報酬の切り分けが困難**
   - 広告運用の成果に対して報酬を設定したい場合、提案Cとの境界が不明確になる
   - 「何に対していくら払っているのか」がクライアントにとって分かりにくい

3. **提案Cの設計を崩す**
   - 提案Cはすでに月43〜50時間という工数で設計されている
   - ここに広告運用（月10時間）を追加すると、料金体系の再設計が必要
   - 11/28の会議を前に、提案内容を変更するのは混乱を招く

**結論**: 選択肢Aは採用しない

---

#### 選択肢B: 別契約として並行

**想定される形**:
- 提案C（情報システム刷新）: 従来通りの内容・料金
- 広告運用サポート: 別契約として月額固定または成果報酬

**メリット**:

1. **価値の明確化**
   - 各契約で「何に対していくら払うか」が明確
   - 提案C: 組織変革・人材育成に対する投資
   - 広告運用: 広告成果に対する投資

2. **成果報酬の設計が可能**
   - 広告契約の中で成果報酬を設計できる
   - 成果の定義・測定方法を明確にできる

3. **既存提案への影響なし**
   - 提案Cの内容を変更する必要がない
   - 11/28の会議では提案Cの判断に集中できる

4. **柔軟性**
   - 広告運用だけ先に開始することも可能
   - 提案Cが不採用でも、広告運用は継続できる

**デメリット**:

1. **契約管理の複雑化**
   - 2つの契約を並行管理する必要がある
   - クライアントの事務負担が増える可能性

2. **総額が大きく見える**
   - 提案C（月21.5〜25万円）+ 広告運用（月5〜6万円）= 月26.5〜31万円
   - 「高い」という印象を与えるリスク

**結論**: 選択肢Bが最も合理的

---

#### 選択肢C: 広告運用を単独で提案

**想定される形**:
- 提案Cとは完全に独立した単独提案
- 提案Cの採否に関わらず、広告運用だけを提案

**問題点**:

1. **関係性の分断**
   - すでに「組織変革パートナー」としてのポジショニングを構築している
   - 広告運用だけを切り出すと、「業者」としての位置づけに後退するリスク

2. **戦略的一貫性の欠如**
   - 提案Cでは「PDCA確立」「業務改善」を謳っている
   - 広告運用も本来はPDCA改善の一部
   - 別々に提案すると、全体の一貫性が見えにくい

**結論**: 選択肢Cは次善策。選択肢Bが不可能な場合の代替案

## 第3章：最適解の導出

### 3-1. 結論

**選択肢B（別契約として並行）を採用する。**

理由:
1. 価値の性質が異なるものを無理に統合しない
2. 成果報酬の設計が可能
3. 既存の提案Cへの影響がない
4. クライアントにとって「何に払っているか」が明確

### 3-2. 広告運用サービスの設計

以下の設計を推奨する。

#### サービス概要

| 項目 | 内容 |
|------|------|
| サービス名 | Web広告運用最適化サポート |
| 対象 | Meta広告（Facebook/Instagram）+ Yahoo広告の見直し |
| 契約形態 | 月額制（成果報酬オプションあり） |
| 契約期間 | 初回3ヶ月、以降1ヶ月更新 |

#### 料金プラン

**プラン1: 固定月額制**

```
月額: 55,000円（税抜）
```

**含まれるサービス**:
- 広告運用の監視・調整（週1回）
- 時間帯・予算配分の最適化
- 月次レポート作成
- 改善提案（月1回）
- MTG（月1回、30分〜1時間）

**プラン2: 成果連動制**

```
基本料金: 33,000円（税抜）/月
成果報酬: 50件超過分 × 2,200円/件（税抜、上限33,000円）
```

**収入シミュレーション**:
| 成果件数 | 基本料金 | 成果報酬 | 合計 |
|---------|---------|---------|------|
| 45件 | 33,000円 | 0円 | 33,000円 |
| 55件 | 33,000円 | 11,000円 | 44,000円 |
| 65件 | 33,000円 | 33,000円 | 66,000円 |

#### 推奨プラン

**成果連動制（プラン2）を推奨する。**

理由:
1. クライアントのリスクを軽減（成果が出なければ基本料金のみ）
2. 成果へのコミットメントを示せる
3. 改善余地が明確なため、成果報酬を得られる可能性が高い
4. 黒澤工務店の意思決定パターン（実利重視、リスク回避的）に合致

## 第4章：提案のタイミングと方法

### 4-1. タイミング

**11/28の会議後に提案する。**

理由:
1. 11/28は提案C（情報システム刷新）の判断日
2. 同時に複数の大型提案をすると、意思決定が分散する
3. 提案Cの採否が決まってから、広告運用を追加提案する方が整理しやすい

### 4-2. 提案シナリオ

**シナリオ1: 提案Cが採用された場合**

「提案Cとは別に、広告運用の改善もお手伝いできます。すでにデータ分析は完了しており、追加費用なしで月10〜20件の成果向上が見込めます。提案Cとは別契約として、月額3.3万円＋成果報酬でいかがでしょうか。」

**シナリオ2: 提案Cが不採用の場合**

「では、広告運用の改善からスタートしませんか。組織変革は大きな投資ですが、広告運用は月3〜5万円程度で始められます。まず成果を出して、信頼関係を深めてから、改めて組織全体のご相談をさせてください。」

### 4-3. 提案時の注意点

1. **提案Cとの関係を明確に説明する**
   - 「別の契約です」と明言する
   - 「何に対していくら」を明確にする

2. **成果報酬の定義を明確にする**
   - CVの定義（購入、予約、問い合わせ）
   - 測定方法（Meta広告管理画面のデータを基準）
   - 計測期間（毎月1日〜末日）

3. **無理に押し込まない**
   - 広告運用は「あれば良い」という位置づけ
   - 提案Cが最優先。広告は追加オプション

## 第5章：工数と収益性の確認

### 5-1. 想定工数

| 作業 | 時間/月 |
|------|--------|
| 広告運用・監視 | 4時間 |
| レポート作成 | 3時間 |
| 改善施策の検討 | 2時間 |
| MTG・コミュニケーション | 1時間 |
| **合計** | **10時間/月** |

### 5-2. 収益性

**プラン2（成果連動制）の場合**:

| シナリオ | 収入 | 工数 | 時間単価 |
|---------|------|------|---------|
| 最低（45件） | 33,000円 | 10時間 | 3,300円 |
| 標準（55件） | 44,000円 | 10時間 | 4,400円 |
| 最高（65件） | 66,000円 | 10時間 | 6,600円 |

改善余地が明確なため、標準〜最高シナリオの実現可能性は高い。

### 5-3. 提案C + 広告運用の総額

提案Cと広告運用を並行した場合の月額:

| 項目 | Phase 1 | Phase 2 | Phase 3 |
|------|---------|---------|---------|
| 提案C | 21.5〜25万円 | 10〜12.5万円 | 4〜5万円 |
| 広告運用 | 3.3〜6.6万円 | 3.3〜6.6万円 | 3.3〜6.6万円 |

Phase 1の総額は高額に見えるが、提供価値（組織変革＋広告改善）を考えれば正当化できる。

## 第6章：結論

### 6-1. 最終結論

**広告運用は、提案Cとは別契約として提案する。**

**推奨プラン**: 成果連動制（基本3.3万円＋成果報酬）

**提案タイミング**: 11/28の会議後

### 6-2. この結論に至った論理

1. **価値の性質が異なる**
   - 提案C: プロセス価値（組織変革、人材育成）
   - 広告運用: 成果価値（コンバージョン改善）
   - 異なる価値を無理に統合すると、評価基準が曖昧になる

2. **成果報酬の設計を可能にする**
   - 広告運用では成果に応じた報酬を設定したい
   - 別契約にすることで、成果の定義・測定・報酬の紐付けが明確になる

3. **既存提案への影響を避ける**
   - 提案Cはすでに設計が完了し、11/28に判断を控えている
   - ここで内容を変更すると混乱を招く

4. **クライアントの意思決定パターンに合致**
   - 黒澤工務店は「何に対していくら」を明確にすることを好む
   - 別契約にすることで、透明性が高まる

### 6-3. 次のアクション

1. 11/28の会議で提案Cの判断を確認する
2. 提案Cの結果に応じて、広告運用の提案タイミングを決定する
3. 広告運用の提案書を作成する（本資料をベースに）
4. 契約条件（成果の定義、計測方法、支払条件）を明文化する

---

## おわりに

この結論は、「パッケージにした方がシンプル」という直感的な考えを否定するものである。

しかし、ビジネスにおいては「シンプルさ」よりも「明確さ」が重要な場面がある。特に、異なる性質の価値を提供する場合、それぞれを明確に分けることで、クライアントとの認識齟齬を防ぎ、長期的な信頼関係を築くことができる。

「別契約」という結論は、一見すると複雑に見えるかもしれない。しかし、それは「何に対していくら払っているか」を明確にするための複雑さであり、双方にとって健全な取引関係を構築するための必要な構造である。
