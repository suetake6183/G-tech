# ブツブツ喋りからタスク抽出プロンプト

## 1. 背景と目的
- **タスクの背景**：頭の中で考えていることを言語化（ブツブツ喋り）したものの、それがまだ整理されておらず、具体的な行動計画に落とし込めていない状態です
- **最終目標**：言語化された未整理の考えから、具体的で実行可能なタスクリストと行動計画を作成すること
- **対象読者**：タスクを自分自身で整理して実行したい個人（自分）

## 2. Claudeへの役割設定
```
あなたはタスク管理、プロジェクト管理、および思考整理の専門家として振る舞ってください。
未整理の思考から具体的なタスクを抽出し、それを実行可能な行動計画に変換することに長けています。
```

## 3. 指示内容
- **実行すべきタスク**：
  1. 私が提供する「ブツブツ喋り」のテキストを注意深く読み、そこに含まれる潜在的なタスクや行動項目をすべて抽出してください
  2. 抽出したタスクを以下の視点で分析・整理してください：
     - 優先順位（重要度と緊急度）
     - 関連性（相互依存関係）
     - 実行難易度
     - 必要なリソース
  3. 不明確な部分や詳細が足りない部分については、明確化のための質問をリストアップしてください
  4. 上記の分析に基づいて、具体的で実行可能なタスクリストと行動計画を作成してください

- **特に注意すべきポイント**：
  - 漠然とした考えや抽象的な表現を、具体的なアクションに変換してください
  - 単に言及されているだけの事項と、実際に行動が必要なタスクを区別してください
  - 複雑なタスクは、より小さな実行可能なステップに分解してください
  - 私が「やりたいこと」と「やるべきこと」を混同している可能性があるため、それらを適切に区別してください

## 4. 出力形式の指定
- **形式**：構造化されたタスクリストと行動計画
- **長さ**：抽出されたタスクの量に応じて適切な長さ（簡潔さと網羅性のバランスを重視）
- **構造**：
  - タスク概要（全体像の把握）
  - カテゴリー別タスクリスト
  - 優先順位別のタスク整理
  - 時系列での行動計画
  - 明確化が必要な事項のリスト

## 5. 例示（マルチショットプロンプト）
### 入力例
```
あー、そうだなあ、週末までにあのレポート終わらせないといけないんだよな。でも資料が足りないかも。田中さんに連絡して追加データもらわないと。あとウェブサイトのリニューアルも考えなきゃ。デザイナーとの打ち合わせも入れないと。そういえば来月の会議の準備も始めないと。誰を招待するか決めないといけないし、場所も押さえないと。ああ、その前に予算確認だ。経理部に確認メール送らないと。あと個人的には、そろそろ運動も始めたいなあ。毎朝10分でもジョギングするとか。健康診断の予約もしなきゃ。
```

### 期待される出力例
```
# タスク分析と行動計画

## タスク概要
提供されたテキストからは、仕事関連のタスクと個人的なタスクの両方が見受けられます。仕事では主にレポート作成、ウェブサイトリニューアル、会議準備が挙げられており、個人的には健康管理が課題となっています。

## カテゴリー別タスクリスト

### 1. レポート関連
- レポートを週末までに完成させる
- 田中さんに連絡して追加データを入手する

### 2. ウェブサイト関連
- ウェブサイトリニューアルの計画を立てる
- デザイナーとの打ち合わせをスケジュールする

### 3. 会議準備関連
- 来月の会議の招待者リストを作成する
- 会議場所を予約する
- 経理部に予算確認のメールを送る

### 4. 個人の健康管理
- 毎朝10分のジョギングを計画に組み込む
- 健康診断の予約をする

## 優先順位別タスクリスト

### 緊急・重要（すぐに対応）
- 田中さんに連絡して追加データを入手する（レポートの締切が週末のため）
- 経理部に予算確認のメールを送る（会議準備の前提条件）

### 重要だが緊急ではない（計画的に対応）
- ウェブサイトリニューアルの計画立案
- 会議の招待者リスト作成と場所の予約
- 健康診断の予約

### 習慣化が必要（定期的に対応）
- 毎朝10分のジョギング

## 時系列での行動計画

### 今日
1. 田中さんに追加データのメールを送信
2. 経理部に予算確認のメールを送信
3. 健康診断の予約を入れる

### 明日
1. 田中さんから受け取ったデータをレポートに統合開始
2. デザイナーとの打ち合わせをスケジュール
3. 朝のジョギングを開始（10分間）

### 明後日～週末
1. レポートの完成と提出
2. 会議の招待者リストのドラフト作成
3. 毎朝のジョギングを継続

### 来週
1. ウェブサイトリニューアルの計画書作成
2. 会議場所の予約確定
3. 朝のジョギングを習慣化

## 明確化が必要な事項
1. レポートの具体的な締切日は？（「週末」とありますが、具体的な日付は？）
2. 追加データの具体的な内容は？
3. ウェブサイトリニューアルの目的・範囲・期限は？
4. 来月の会議の正確な日程と目的は？
5. 経理部に確認する予算の具体的な項目や金額は？
6. 健康診断の希望時期や受診したい医療機関は？
```

## 6. 評価基準
以下の基準を満たす回答を生成してください：
- すべての潜在的なタスクが漏れなく抽出されていること
- 各タスクが具体的で行動可能な形式になっていること
- タスク間の関連性や依存関係が明確になっていること
- 優先順位が適切に設定されていること
- 実行するための時間軸が明確になっていること
- 不明確な点が適切に指摘され、具体化のための質問が提示されていること

## 7. XMLタグ構造
```xml
<input>
[ブツブツ喋りのテキストをここに貼り付けてください]
</input>

<analysis>
[テキストからの初期分析と全体的な印象]
</analysis>

<tasks>
[抽出されたタスクのリスト（カテゴリー別・優先順位別）]
</tasks>

<plan>
[時系列での具体的な行動計画]
</plan>

<questions>
[明確化が必要な事項についての質問]
</questions>
```

## 使用方法
1. 上記の `<input>` タグ内に、あなたが頭の中で考えていることを言語化した「ブツブツ喋り」のテキストを貼り付けます
2. Claudeが提供したテキストを分析し、具体的なタスクと行動計画を提示します
3. 明確化が必要な事項について、Claudeからの質問に回答することでタスクをさらに具体化します
4. 必要に応じて、生成された行動計画をさらに詳細化するようClaudeに依頼することもできます
