# リフォーム工事業務フローチェックリスト

## 1. 契約前段階

### 初回相談・現地調査
- [ ] 顧客のニーズと予算のヒアリング実施
- [ ] 現地調査の実施と状況記録（写真撮影）
- [ ] 工事に影響する建物の構造的問題の確認
- [ ] 法的制限（建築基準法、消防法など）の確認
- [ ] 予想されるリスク要因の特定（古い建物の場合の隠れた劣化など）

### 見積り・提案
- [ ] 詳細な見積書の作成（使用材料、工法、機器などを明記）
- [ ] 標準工事と追加オプションの明確な区分
- [ ] 複数の工事プラン・選択肢の提示（可能な場合）
- [ ] 予想される追加費用発生リスクの説明
- [ ] 見積有効期限の明示

### 契約準備
- [ ] 工事内容の詳細仕様書の作成
- [ ] 使用材料・設備のカタログ・サンプルの準備
- [ ] 工期の算定と工程表の作成
- [ ] 支払条件の設定
- [ ] クーリングオフ対象か否かの確認

## 2. 契約締結段階

### 重要事項説明
- [ ] 工事内容と仕様の詳細説明
- [ ] 代金額と支払条件の説明
- [ ] 工期と引渡し時期の説明
- [ ] 契約不適合責任の内容と期間の説明
- [ ] 追加・変更工事の取扱いについての説明
- [ ] クーリングオフ制度の説明（該当する場合）

### 契約書作成・締結
- [ ] 顧客情報の正確な記入
- [ ] 工事内容・仕様の詳細記載
- [ ] 代金額・支払条件の明記
- [ ] 工期の明記
- [ ] 添付書類（仕様書、図面、カタログなど）の確認
- [ ] 契約書への署名・捺印（電子契約の場合は適切な認証）
- [ ] 契約書と重要書類の顧客への交付

## 3. 着工準備段階

### 社内準備
- [ ] 工事スケジュールの確定と人員配置
- [ ] 必要な材料・設備の発注
- [ ] 下請業者との契約・スケジュール調整（必要な場合）
- [ ] 工事保険への加入確認
- [ ] 施工図面・詳細施工計画の作成

### 顧客との最終確認
- [ ] 着工日の最終確認
- [ ] 工事中の生活上の注意点説明
- [ ] 家財の移動・保護についての打合せ
- [ ] 工事中の連絡方法の確認
- [ ] 近隣への挨拶・説明の実施

## 4. 工事実施段階

### 着工時
- [ ] 着工前の現状写真撮影
- [ ] 養生の実施
- [ ] 工事内容の最終確認
- [ ] 安全対策の確認

### 工事進捗管理
- [ ] 日々の工事内容と進捗の記録
- [ ] 定期的な顧客への進捗報告
- [ ] 工事写真の撮影（特に隠蔽部分の記録）
- [ ] 予定工程との差異の確認と対応
- [ ] 品質管理・検査の実施

### 追加・変更対応
- [ ] 追加・変更工事の発生確認
- [ ] 顧客への書面による説明と見積提示
- [ ] 追加・変更工事の承諾書取得
- [ ] 工期への影響の確認と調整
- [ ] 契約書への追加・変更内容の反映

### 中間金請求（該当する場合）
- [ ] 中間金支払条件の達成確認
- [ ] 中間検査の実施と記録
- [ ] 請求書の発行
- [ ] 入金確認

## 5. 完成・引渡し段階

### 完成検査
- [ ] 自社による完成前検査の実施
- [ ] 不具合箇所の修正
- [ ] 設備等の動作確認
- [ ] 清掃の実施

### 顧客立会い検査
- [ ] 顧客立会いのもとでの完成検査の実施
- [ ] 工事内容の説明
- [ ] 設備の使用方法の説明
- [ ] 検査内容の記録

### 引渡し手続き
- [ ] 完成引渡書の作成と署名取得
- [ ] 取扱説明書等の関連書類の引渡し
- [ ] 残金の請求と受領
- [ ] 保証書の発行
- [ ] アフターサービス内容の説明

## 6. アフターフォロー段階

### 初期フォロー
- [ ] 引渡し後1ヶ月以内の顧客訪問または連絡
- [ ] 不具合の有無確認
- [ ] 軽微な調整等の実施（必要な場合）
- [ ] 顧客満足度の確認

### 契約不適合責任対応
- [ ] 不適合の申し出への迅速な対応
- [ ] 現地調査と原因分析
- [ ] 修補または代替措置の提案
- [ ] 対応完了の確認と記録

### 長期フォロー
- [ ] 定期的なメンテナンス案内
- [ ] 季節に応じた使用上の注意喚起
- [ ] 追加リフォームの提案（適切な時期に）
- [ ] 顧客関係の維持・強化

## 7. 記録管理

### 工事記録
- [ ] 契約書類一式の保管
- [ ] 工事写真の整理・保存
- [ ] 打合せ記録の保存
- [ ] 資材・設備の情報記録

### 顧客情報管理
- [ ] 顧客情報の適切な管理（個人情報保護）
- [ ] 保証期間の管理
- [ ] フォローアップスケジュールの設定
- [ ] 顧客の声・評価の記録

---

**注意事項：**
- このチェックリストは一般的なリフォーム工事を想定しています。工事の規模や内容に応じて適宜調整してください。
- 地域や工事内容によって法規制が異なる場合があります。必ず最新の法的要件を確認してください。
- 消費者との契約の場合は、特に消費者保護関連法規に留意してください。
