## ⚠️ リスクと対策

### リスク1: 「AI使うなら、もっと安くできるんじゃ？」

**確率**: 20%

**対応**:
```
「確かにAI使うと早いんだけど、
AIが作ったコードって、そのままじゃ使えないんですよ。

例えば、AIに「集計して」って言ったら、
一般的な集計コードは作ってくれるけど、
DOAさん専用のカスタマイズは人間がやらないといけない。

AI: 4時間（基本コード生成）
人間: 6時間（カスタマイズ・調整・テスト）
合計: 10時間

だから、工数が1/3になっても、ゼロにはならないんです」
```

### リスク2: 「8万円でも高い」

**確率**: 30%

**対応**:
```
「分かります。でも考えてみてください。

今、毎日30分の転記作業してますよね？
月20日として、月10時間。
年間120時間。

雅さんの時給を2,000円としたら、
年間24万円分の時間が節約できます。

8万円の投資で、年間24万円の効果。
3ヶ月で元が取れる計算です。

しかも転記ミスもなくなるし、
どこからでもデータ見られるようになる。

それでもまだ高いですか？」
```

**代替案**:
- 分割払いの提案（月2万円×4ヶ月）
- 機能を削ってさらに値下げ（6万円程度）

### リスク3: 「Phase 1だけでいい」と言われる

**確率**: 40%

**対応**:
```
「もちろん、それでも大丈夫です！

Phase 1だけでも、
場内のデータが記録できるようになるし、
ネット環境も良くなります。

で、それを使ってみて、
『やっぱり転記作業面倒だな』
って思ったら、その時にPhase 2やりましょう。

無理に今決めなくていいですよ」
```

**内心の戦略**:
- Phase 1で満足度を高める
- Phase 1完了後、自然にPhase 2の話をする
- 焦らない、長期的な関係を重視
