**総合**: ⭐⭐ / ⭐⭐⭐⭐⭐（初心者レベル）

| 項目 | レベル | 証拠 |

| クラウド概念 | ⭐ | Dropbox・Googleスプレッドシート未使用 |

### 🚨 最重要発見：「1ヶ月前まで完全手書き」

**2024年9月まで**:
- 日報：完全手書き
- 計算：電卓
- 集計：手作業

**2024年10月〜現在**:
- タブレット + キングソフト（Excel互換）導入
- デジタル化開始から**わずか1ヶ月**
- 「ちょっと慣れてきたかなみたいな」（雅さん）

**この情報の意味**:
- ✅ 変更への抵抗が低い（まだ慣れていない）
- ✅ どんな改善でも「画期的」に感じる
- ✅ 期待値が低い（満足度を高めやすい）
- ⚠️ ITリテラシーが非常に低い
- ⚠️ トラブル時の対応が困難
- ⚠️ 複雑なシステムは使いこなせない

### 現在の業務フロー

**使用ソフトウェア**:
- キングソフト（中国製のExcel互換、安価版）
- 正式なExcelではない

**使用デバイス**:
- メイン：2万円のタブレット（1年半前購入、CPU性能極めて低い）
- サブ：Surfaceっぽいパソコン（菅野さんから借用、未使用）

**運用方法**:

- 日ごとにシートを作成
- 日報 → 小計表 → 売掛台帳への転記は**全て手作業**

**できること**:
- 日報への手入力
- 簡単な合計計算（SUM関数程度）
- 基本的な表作成

**できないこと**:
- 複雑な関数
- データ連携・自動転記
- クラウド活用
- マクロ・自動化
- 複数デバイスでの同時作業

## 対応スタンス・戦略

### 基本方針

**「友人」と「プロ」のバランス**

```
友人として:
- 気軽に相談できる関係を維持
- 背中を押してあげる
- 長期的な成功を願う

プロとして:
- 適正な対価をいただく
- 品質を保証する
- 継続的なサポートを提供
```

### 価格戦略

**元同僚価格 = 市場相場の40-50%**

理由:
- **17年来の元同僚・戦友**という特別な関係
- 同じ店で働いた仲間への支援
- 元職場（DOA）の成功を本気で応援したい
- でも「安すぎて適当にやってる」と思われるのは避ける
- プロとしての品質は絶対に守る

**価格設定の哲学**:
```
市場価格: 20-30万円（完全なビジネス）
友人価格: 15-20万円（6年来の友人）
元同僚価格: 10万円（17年来の戦友）← 今回

「お前の成功を本気で応援したいから、
ギリギリまで安くした。
でも品質は絶対に妥協しない」
```

### コミュニケーション戦略

1. **まず対面で会う**（お店訪問）
2. **図を描きながら説明**
3. **「まず2.5万円でこれやってみよう」と提案**
4. **その場で方向性を決める**
5. **後から簡単な見積書を送る**

### リスク管理

**友人関係を壊さないために**:
- ❌ いきなり高額請求しない
- ✅ 段階的に進める
- ✅ 常に相談しながら進める
- ✅ 無理強いしない
- ✅ 失敗したら責任を持つ
