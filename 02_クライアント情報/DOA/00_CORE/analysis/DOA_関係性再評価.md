## 🎯 価格戦略の再評価

```
【市場相場】
システム開発会社: 30-50万円
フリーランス上級: 20-30万円
フリーランス中級: 10-20万円

【今回の価格】

【比較】
市場相場の下限: 10万円

→ 実は「友人価格」ではなく「戦友価格」
```

### 価格設定の哲学（改訂版）

```
完全ビジネス価格: 30万円（他人）
   ↓ -33%
ビジネス友人価格: 20万円（6年来の友人）
   ↓ -50%
元同僚価格: 10万円（17年来の戦友）← 今回
   ↓ さらに下げるべき？
戦友無償サポート: 0円（できるけどやらない）

理由: プロとしての品質を守るため、
     適正な対価は必要
```

## 🤔 さらに値下げすべきか？

### Option A: 現状維持（10万円）

**メリット**:
- ✅ 適正な時給（4,000円）を維持
- ✅ プロとしての品質を担保
- ✅ 「安すぎて適当にやってる」と思われない

**デメリット**:
- ⚠️ 17年来の元同僚にしては高い？

### Option B: さらに値下げ（7-8万円）

**Phase 2を80,000円 → 60,000円に変更**

```
Phase 1: 25,000円

─────────────

```

**メリット**:
- ✅ より「元同僚価格」を実感してもらえる
- ✅ 成約確率がさらに向上（90-95%）
- ✅ 「お前のためを思って最安値にした」

**デメリット**:
- ❌ 実質時給が3,100円程度に下がる
- ❌ 利益が薄すぎる

### Option C: 無償サポート（部分的に）

**Phase 1を25,000円 → 無償に変更**

```
Phase 1: 無償（元同僚へのプレゼント）
Phase 2: 80,000円
─────────────
合計: 80,000円
```

**メリット**:
- ✅ 「元同僚への支援」が明確
- ✅ Phase 1で完全な信頼構築
- ✅ Phase 2への移行がスムーズ

**デメリット**:
- ❌ 「無償でやってくれる」と思われる
- ❌ 後から有償に切り替えにくい

## ✅ 推奨：Option A（現状維持 10万円）

### 理由

1. **プロとしての姿勢**
   ```
   「お前のことは17年の付き合いだから、
   ギリギリまで安くした。
   でも、プロとして品質は絶対に妥協しない。
   だから、最低限の対価はいただく」
   ```

2. **長期的な関係**
   - 今回無償にすると、次回も無償を期待される
   - 適正な対価をもらうことで、継続的な関係が築ける

3. **雅さんのためでもある**
   - 安すぎると「適当にやってる」と不安になる
   - 適正価格 = プロの仕事 = 安心感

4. **市場相場との整合性**
   - 10万円でも市場相場の下限
   - これ以上下げると「怪しい」
