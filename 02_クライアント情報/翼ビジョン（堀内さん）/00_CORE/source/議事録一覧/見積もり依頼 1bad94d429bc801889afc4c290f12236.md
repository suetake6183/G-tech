# 見積もり依頼

日付: 2025/03/18
プロジェクト: - 🏢 店舗スペースの概要 約66㎡の居抜き物件を飲食店として利用予定。
- 🔧 主要工事内容 内装リフォームや水回り設備の更新、厨房設備工事が含まれる。
- 🚧 推奨工事項目 防水・防音対策や安全対策、店舗演出に関する工事も提案。
- ⚠️ 潜在的問題点 給排水配管の老朽化や電気容量不足などのリスクに対策を講じる必要がある。
- 💰 利益率確保のポイント 材料選定や工事手法を工夫し、利益率20%以上を目指す。

## 1. 現状分析と工事概要

- **物件概要**: 約66㎡（20坪）の店舗スペース（居抜き物件を想定）
- **用途**: 飲食店（カフェまたはレストラン）
- **主要工事**: 内装リフォーム、水回り設備更新、厨房設備工事
- **予算条件**: 銀行融資用の仮想見積もり（実際の物件は未定）
- **基本方針**: 内装のみで外装・フロント工事なし、既存間取りを変更

## 2. 主要工事項目（必須）

### 解体・撤去工事

- 既存内装材（壁・天井）解体撤去
- 既存床材撤去（上貼り施工のため必要に応じて）
- 残置物撤去処分
- 業務用エアコン（室外機）撤去
- 既存設備（トイレ、洗面台）撤去

### 内装工事

- 床材上貼り施工（図面のタイル柄床材）
- 壁クロス張替え（1000番台）
- 天井クロス張替え
- トイレ区画工事（男女別トイレ設置）
- 間仕切り工事（図面に基づく）
- 出入口ドア新設
- アーチ壁仕上げ（参考イメージ写真のような仕様）

### 電気設備工事

- 電気幹線引込工事
- 分電盤新設・更新
- コンセント新設（厨房機器用強化）
- 照明器具設置（図面の青線表示箇所）
- ダウンライト設置（図面のピンク丸表示箇所）
- ペンダントライト設置（図面の黄色丸表示箇所）
- Wi-Fi設備新設
- 防犯カメラ設置（必要に応じて）

### 給排水設備工事

- 給水・給湯配管工事
- 排水管工事（厨房・トイレ・洗面所）
- 給湯器設置
- トイレ便器新設（2台）
- 洗面台設置（3か所）
- 手洗い・自動水栓設置（図面表示箇所）
- グリストラップ設置（または既存清掃・点検）

### 空調設備工事

- 業務用エアコン新設（室内機・室外機）
- ダクト工事
- 換気設備工事
- パッケージエアコン設置（図面記載の通り）

## 3. 二次的工事項目（推奨）

### 防水・防音対策

- トイレ・洗面エリア防水工事

- 換気扇・フード設置工事

### 外構・その他

## 4. 潜在的問題点と対応策

### 給排水関連

- **問題点**: 既存配管の老朽化・腐食の可能性
- **対策**: 事前調査と配管更新範囲の明確化、予備費計上

### 電気容量

- **問題点**: 厨房機器増設による電気容量不足リスク
- **対策**: 電気容量計算と必要に応じた幹線・分電盤の増強

### 換気・空調

- **問題点**: 厨房からの臭気拡散、空調効率の問題
- **対策**: 適切な換気設計、厨房と客席の空調分離

### 防火対策

- **問題点**: 消防法規制への対応不足
- **対策**: 防火区画・内装制限の確認、消防設備の適切な設置

### 施工スケジュール

- **問題点**: 工期遅延リスク（特に設備工事の納期）
- **対策**: 資材の先行発注、工程表の綿密な作成

## 5. 利益率確保のためのポイント

### 材料選定

- 国内メーカー製ペンダントライト（25,000円/個程度）を提案（粗利率30〜35%）
- 床材・壁紙は当社指定メーカー品（粗利率25%以上）を使用
- トイレ設備は標準グレードでパッケージ提案（粗利率30%）

### 工事手法

- 床は既存の上に上貼り工法を採用（解体費削減）
- パッケージエアコンは性能・価格のバランスが良い機種を選定
- 電気工事は将来の増設を考慮した余裕ある設計（追加工事の可能性）

### 契約方法

- 材料費と工事費を分離した見積書作成（材料の値上がり対策）
- 追加工事発生時の単価をあらかじめ設定
- 工事完了検査時の追加工事提案を計画

### 工程管理

- 専門工事の外注比率を適正化（自社施工で利益確保）
- 工期短縮による経費削減（特に仮設費、現場管理費）
- 資材発注の一括化によるボリュームディスカウント

### サービス付加価値

- 施工後のメンテナンスプランを提案（定期点検契約）
- 厨房機器の中古仕入れサポート（紹介料確保）
- 店舗オープン前の清掃サービス（オプション工事）

## 6. 確認・質問事項

1. 厨房の排気・排煙設備の詳細（調理内容によって設備が変わる）
2. 電気容量の確認（厨房機器の仕様にあわせた容量計算が必要）
3. 営業開始予定時期（工期設定の参考に）
4. 家具・什器の搬入タイミング（工程調整のため）
5. 看板・サイン計画の有無（電源位置の検討）
6. 周辺環境の規制（騒音・臭気対策の必要性）
7. グリストラップの設置要否（自治体の規制確認）
8. 中古厨房機器のサイズ・仕様（設置スペース確保）
