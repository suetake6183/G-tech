# BIZ_業種別クライアント

## 概要
業種ごとにクラスタリングされた継続取引先です。
各業種特有のビジネスモデルやニーズに対応したサービスを提供しています。

## 業種別カテゴリ

### 建設・リフォーム
建設・リフォーム業界のクライアント群。相互に案件紹介や協業を行う強力なネットワークを形成。

**クライアント**:
- **SKコーム**: 自社（株式会社SKコーム、代表：末武修平）
- **ジューテクノ**: リフォーム仲介、案件紹介パートナー
- **柳屋関連**: 不動産×リフォームの協業案件

**特徴**:
- 業界内での相互紹介、施工協力
- 技術情報の共有
- 案件規模：中小規模のリフォーム工事が中心

---

### 不動産管理
不動産管理会社からの定期的な空室対策工事発注。

**クライアント**:
- **青柳不動産**: 空室工事の定期発注
- **アストライズ**: 物件メンテナンス
- **エトワール**: 物件別管理（山崎物件等）

**特徴**:
- 定期的な発注パターン
- 年度別フォルダ管理（2024/2025）
- 見積・請求書の継続的な蓄積

---

### 店舗・飲食
店舗の看板工事、内装工事、システム支援。

**クライアント**:
- **AOMI100クリニック**: 看板・内装工事（高額案件）
- **その他進行中プロジェクト**: うらいち店舗、こぶしマンション等

**特徴**:
- 単発の大型プロジェクトが多い
- 看板、内装、設備の複合工事
- プロジェクトごとの詳細管理

---

### Web・IT
Web制作、IT支援サービス。

**クライアント**:
- **WEB屋喜作**: 継続的なWeb制作案件
- **樋口様**: 法律事務所のWebサイト制作（写真素材提供）

**特徴**:
- デザイン・制作物の管理
- 写真素材、ロゴデータの保管
- 請求書管理

## 管理方針
1. **業種別のベストプラクティス共有**: 同業種での成功事例を横展開
2. **定期的なフォローアップ**: 継続取引を維持するための定期連絡
3. **年度別管理**: 不動産管理等は年度別フォルダで整理
4. **案件別管理**: 店舗・飲食等は案件別フォルダで整理

## データ構造
継続取引先は年度別、スポット案件は案件別での管理を推奨。
各クライアントの特性に応じて柔軟に対応。
