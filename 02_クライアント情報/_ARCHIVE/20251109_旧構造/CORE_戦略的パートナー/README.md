# CORE_戦略的パートナー

## 概要
最も関係性が深く、長期的な戦略的パートナーシップを構築しているクライアント群です。
単なる取引先ではなく、相互の成長を目指す協業関係にあります。

## 特徴
- **深い信頼関係**: 経営課題の共有、戦略的な意思決定への関与
- **複合的なサービス**: 施工だけでなく、コンサルティング、AI/DX支援、システム構築など
- **長期的コミットメント**: 継続的な伴走支援、定期的な戦略会議
- **相互成長**: お互いのビジネス成長を支援し合う関係

## クライアント一覧

### 黒澤工務店
- **関係性**: 13ヶ月の伴走支援、AI導入・DX推進パートナー
- **サービス**: 業務提携、情報システム構築、AI活用支援、施工協力
- **特記事項**: 最も情報密度が高いクライアント。ナレッジベース構築、詳細な分析レポート

### DOA（ホストクラブ）
- **関係性**: 17年来の元同僚、システム化支援パートナー
- **サービス**: システム構築、業務効率化支援、Phase 1/2提案
- **特記事項**: 深い個人的信頼関係に基づく長期プロジェクト

### 翼ビジョン（堀内さん）
- **関係性**: インテリアコーディネート×施工の協業パートナー
- **サービス**: AI活用支援、デザイン×施工の協業案件
- **特記事項**: 5ヶ月の空白期間を経て関係再構築中

## 管理方針
1. **定期的なコミュニケーション**: 月次または四半期ごとの戦略会議
2. **詳細な記録**: 会議議事録、戦略文書、分析レポートを丁寧に保管
3. **ナレッジ蓄積**: 学びや気づきをナレッジベース化
4. **プロアクティブな提案**: 相手のビジネス課題を先回りして解決策を提案

## データ構造の推奨
各クライアントフォルダ内は、以下の構造を推奨：
```
クライアント名/
├── 01_プロジェクト/         # プロジェクト別フォルダ
├── 02_コミュニケーション記録/ # 会議録、LINE履歴など
├── 03_ナレッジベース/       # 学びや気づきの蓄積
├── 04_契約・請求/           # ビジネス文書
└── 99_アーカイブ/           # 完了プロジェクト
```
