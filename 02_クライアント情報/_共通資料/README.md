# _共通資料（共通リソース管理）

## 🎯 このフォルダの目的

複数のクライアント・プロジェクトで**再利用可能な素材、テンプレート、知識**を一元管理します。

---

## 📁 フォルダ構造

```
_共通資料/
├── INPUT_LIBRARY/              # プロジェクトで使う素材・参考情報
├── OUTPUT_TEMPLATES/           # プロジェクトで生成する時の雛形
├── KNOWLEDGE_BASE/             # 再利用可能な知識・ノウハウ
└── _ARCHIVE/                   # 過去のもの
```

---

## 🔄 プロジェクトとの関係

### プロジェクト構造との対応
```
【プロジェクト】              【共通資料】
00_INPUT/          ←参照←   INPUT_LIBRARY/
01_OUTPUT/         ←参照←   OUTPUT_TEMPLATES/
（知識を活用）      ←参照←   KNOWLEDGE_BASE/
```

### 使い方の例

#### 新規プロジェクトで見積書を作る時
1. `OUTPUT_TEMPLATES/見積書/` からテンプレートをコピー
2. プロジェクトの `01_OUTPUT/クライアント向け/見積書/` に配置
3. プロジェクト固有の情報を記入

#### ロゴが必要な時
1. `INPUT_LIBRARY/ブランド素材/SKコームロゴ/` からコピー
2. プロジェクトの `00_INPUT/ロゴ・デザイン素材/` に配置

#### プロンプトの書き方を学びたい時
1. `KNOWLEDGE_BASE/プロンプト集/` を参照
2. プロジェクトに応用

---

## 📋 各フォルダの詳細

### INPUT_LIBRARY/
プロジェクトのINPUTとして使う素材置き場

- **ブランド素材/**: SKコーム、ウェブフリなどのロゴ、ブランドガイドライン
- **メディア素材/**: 汎用的な写真、動画、サムネイル
- **参考資料/**: 技術資料、業界資料、参考ドキュメント

### OUTPUT_TEMPLATES/
プロジェクトのOUTPUTを作る時のテンプレート

- **契約書/**: 契約書の雛形
- **見積書/**: 見積書の雛形
- **請求書/**: 請求書の雛形
- **プレゼン資料/**: プレゼンテンプレート

### KNOWLEDGE_BASE/
プロジェクト横断的な知識・ノウハウ

- **プロンプト集/**: AIプロンプトのサンプル、ベストプラクティス
- **マニュアル/**: 業務マニュアル、ガイドライン
- **営業ノウハウ/**: 営業トーク、提案手法、成功事例
- **教材/**: 研修資料、教育コンテンツ

### _ARCHIVE/
古いファイル、使わなくなったもの

---

## ⚠️ 重要なルール

### ✅ ここに入れるもの
- 複数のクライアント・プロジェクトで使う可能性があるもの
- テンプレート、雛形
- 汎用的な素材
- 再利用可能な知識・ノウハウ

### ❌ ここに入れないもの
- **特定クライアント固有のもの** → 各クライアントフォルダへ
- **特定プロジェクト固有のもの** → 各プロジェクトフォルダへ
- **個人的なデータ** → 個人フォルダへ
- **古い・使わないもの** → _ARCHIVE/へ

---

## 🗂️ ファイル命名規則

### テンプレートファイル
- `[種類]_テンプレート_[バージョン].拡張子`
- 例: `契約書_テンプレート_v2.docx`

### 素材ファイル
- そのまま使える名前
- 例: `SKコーム_ロゴ_正方形.ai`

### ドキュメントファイル
- `[カテゴリ]_[内容].md`
- 例: `プロンプト_スライド資料作成.md`
