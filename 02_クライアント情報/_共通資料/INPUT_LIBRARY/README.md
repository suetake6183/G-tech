# INPUT_LIBRARY（インプット素材ライブラリ）

## 📥 このフォルダの役割

プロジェクトのINPUTとして使う**素材・参考情報**を格納します。

---

## 📁 サブフォルダ

### ブランド素材/
SKコーム、ウェブフリなどのロゴ、ブランドガイドライン

**内容例:**
- ロゴデータ（AI, SVG, PNG）
- ブランドカラー定義
- フォント指定

**使い方:**
プロジェクトで会社ロゴが必要な時、ここからコピーして `00_INPUT/ロゴ・デザイン素材/` に配置

### メディア素材/
汎用的な写真、動画、サムネイル

**内容例:**
- 施工例の写真
- 参考デザイン画像
- 説明用動画

**使い方:**
提案資料や報告書で使う汎用的な画像・動画を保管

### 参考資料/
技術資料、業界資料、参考ドキュメント

**内容例:**
- 施工マニュアル
- 業界レポート
- 法規制資料

---

## 💡 使い方のコツ

1. **プロジェクトにコピーして使う**
   - このフォルダから直接編集しない
   - プロジェクトフォルダにコピーしてから編集

2. **元データは保持**
   - オリジナルはここに残す
   - バージョン管理が必要なら `_v1`, `_v2` を付ける

3. **汎用的なもののみ**
   - 特定クライアント専用のものは入れない
   - 複数プロジェクトで使えるものだけ

---

## ❌ ここに入れないもの

- クライアントから提供された素材 → プロジェクトの `00_INPUT/` へ
- プロジェクト固有の写真・資料 → プロジェクトの `00_INPUT/` へ
- 古い素材 → `_ARCHIVE/` へ
