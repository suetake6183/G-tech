# OUTPUT_TEMPLATES（アウトプットテンプレート）

## 📤 このフォルダの役割

プロジェクトのOUTPUTを作る時の**雛形・テンプレート**を格納します。

---

## 📁 サブフォルダ

### 契約書/
契約書の雛形

**内容例:**
- 工事請負契約書テンプレート
- 業務委託契約書テンプレート
- 秘密保持契約書テンプレート

**使い方:**
1. テンプレートをプロジェクトフォルダの `01_OUTPUT/クライアント向け/` にコピー
2. クライアント情報、プロジェクト詳細を記入
3. 最終版を保存

### 見積書/
見積書の雛形

**内容例:**
- 見積書テンプレート（縦型・横型）
- 工事別見積フォーマット
- 明細書テンプレート

**使い方:**
1. テンプレートをプロジェクトフォルダの `01_OUTPUT/クライアント向け/見積書/` にコピー
2. 工事内容、金額を記入
3. PDF化して提出

### 請求書/
請求書の雛形

**内容例:**
- 請求書テンプレート
- 領収書テンプレート

**使い方:**
1. テンプレートをプロジェクトフォルダにコピー
2. 請求内容を記入
3. 送付

### プレゼン資料/
プレゼンテンプレート

**内容例:**
- 提案書テンプレート
- スライドデザインテンプレート
- 会社案内テンプレート

---

## 💡 使い方のコツ

### バージョン管理
- テンプレートにバージョン番号を付ける
- 例: `契約書_テンプレート_v2.docx`
- 古いバージョンは `_ARCHIVE/` へ

### 命名規則
- `[種類]_テンプレート_[バージョン].拡張子`
- 例: `見積書_テンプレート_v1.xlsx`

### プロジェクトでの使い方
1. **テンプレートをコピー**
   ```
   OUTPUT_TEMPLATES/見積書/見積書_テンプレート_v2.xlsx
   ↓ コピー
   プロジェクト/01_OUTPUT/クライアント向け/見積書/20251108_見積書_クライアント名.xlsx
   ```

2. **日付とクライアント名を追加**
   - `YYYYMMDD_[種類]_[クライアント名].拡張子`

3. **内容を記入**

4. **プロジェクトフォルダに保存**
   - テンプレートは上書きしない

---

## ⚠️ 重要なルール

### ✅ ここに入れるもの
- 空白の雛形
- 汎用的なフォーマット
- 複数プロジェクトで使えるテンプレート

### ❌ ここに入れないもの
- クライアント情報が入った完成版 → プロジェクトの `01_OUTPUT/` へ
- 特定クライアント専用のフォーマット → クライアントフォルダへ
- 古いバージョン → `_ARCHIVE/` へ

---

## 🔄 更新ルール

### テンプレートを改善した時
1. バージョン番号を上げる（v1 → v2）
2. 古いバージョンを `_ARCHIVE/` へ移動
3. README.mdに変更履歴を記載

### 新しいテンプレートを追加した時
1. 適切なサブフォルダに配置
2. ファイル名に `_テンプレート_v1` を付ける
3. README.mdに追加
