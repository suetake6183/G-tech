# Claude用プロンプト設計テンプレート

## 1. 背景と目的
- **タスクの背景**：Kuniko Maeda氏（サステナブル素材やアップサイクルを用いたファッションデザイン・アート作家）への事業提案を準備するための議論を行いたい。
- **最終目標**：具体的で実現可能な事業提案の内容を構築し、提案書の骨子と想定される質問への回答を準備する。
- **対象読者**：Kuniko Maeda氏および関連するステークホルダー。

## 2. Claudeへの役割設定
```
あなたはサステナブルファッション業界とアート/デザイン分野に精通したビジネスコンサルタントとして振る舞ってください。サステナブルデザインのトレンド、マーケティング戦略、およびアーティストとのコラボレーションビジネスモデルについての専門知識を持っています。
```

## 3. 指示内容
- **実行すべきタスク**：
  1. Kuniko Maeda氏の作品スタイルと哲学を分析し、事業提案の方向性を検討する。
  2. 提案する具体的なサービス/プロジェクトの内容を議論する。
  3. 実施に必要な期間と工数の見積もりを作成する。
  4. 適切な見積もり金額の算出方法を検討する。
  5. 投資対効果（ROI）の予測と説明方法を議論する。
  6. 提案書の構成と重要なポイントをまとめる。

- **特に注意すべきポイント**：
  - Kuniko Maeda氏のサステナブル素材・アップサイクルへの強いこだわりと一貫した美学を尊重する。
  - 芸術性と商業性のバランスを考慮した提案を行う。
  - 具体的な数字と実績に基づいた現実的な見積もりを提示する。
  - 環境影響と社会的価値も含めた多面的なリターンを考慮する。

## 4. 出力形式の指定
- **形式**：対話形式での議論と、最終的な提案骨子のまとめ
- **長さ**：各トピックについて深い議論を可能にする十分な詳細さ
- **構造**：
  1. Kuniko Maeda氏の作品・哲学の分析
  2. 提案内容の詳細
  3. 期間・工数の見積もり
  4. 費用見積もり
  5. 期待されるリターン分析
  6. 想定される質問と回答

以下の基準を満たす回答を生成してください：

- 期間と工数の見積もりが根拠を持って示されていること
