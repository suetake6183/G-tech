# 新規クライアント追加の手順

## ステップ1: フォルダ構造をコピー

1. `_TEMPLATE` フォルダをコピー
2. クライアント名にリネーム（例: `アストライズ`）

## ステップ2: 年度フォルダを作成

```
[クライアント名]/
├── README.md
├── 2024/
├── 2025/
└── _緊急/          # 忙しい時の一時保管場所
```

## ステップ3: README.mdを編集

- 基本情報を記入
- よく使う単価を記入
- 注意点を追記

## ステップ4: プロジェクトフォルダの作り方

案件が発生したら:

```
[クライアント名]/2025/
└── YYYYMMDD_[案件名]/
    ├── 見積/
    ├── 契約/
    ├── 図面/
    ├── 写真/
    │   ├── 施工前/
    │   └── 施工後/
    └── 請求/
```

---

## フォルダ活用のコツ

### _緊急フォルダの使い方
忙しい時や、どこに保存すべきか迷った時:
1. とりあえず `_緊急/` に保存
2. 週末にまとめて整理

### _共通資料フォルダの使い方
特定のクライアントに紐づかない資料:
- 業界の一般資料
- 技術資料
- 建築基準など

→ `/02_クライアント情報/_共通資料/` に保存

---

## 命名規則（簡略版）

### クライアントフォルダ外（ダウンロード等）
```
YYYYMMDD_クライアント_カテゴリ_内容.拡張子
例: 20250104_アストライズ_見積_池田山.pdf
```

### クライアントフォルダ内
```
YYYYMMDD_カテゴリ_内容.拡張子
例: 20250104_見積_池田山.pdf （クライアント名省略OK）
```

**完璧を目指さない**: 80%守れればOK

---

## 週次レビュー（推奨）

### 毎週金曜日 17:00（15分）

1. ダウンロードフォルダを確認
2. 新しいファイルをクライアント別に移動
3. 命名規則に従ってリネーム（できる範囲で）
4. わからないものは `_緊急/` へ

**完璧主義は禁物**: 忙しい週はスキップしてOK

---

## Claude Code での活用方法

### 見積作成時
1. クライアントフォルダを Claude Code で開く
2. 「README.mdを参照して、新しい見積を作成して」
3. 過去の単価や注意点を理解した提案が生成される

### プロジェクト完了時
1. 「このプロジェクトの学びを README.md に追加して」
2. ナレッジが自動的に蓄積される
